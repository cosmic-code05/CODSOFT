# CODSOFT
Machine Learning Internship Projects by CodSoft.
This repository contains 3 projects: 
1. Movie Genre Classification
2. Spam SMS Detection
3. Handwritten Text Generation (LSTM)

Built using Python, Scikit-learn and TensorFlow.
Each project demonstrates practical implementation of different ML concepts including text processing, classification, and sequence modeling.

---------------------------------------------------------------------------------------------------------------------------------------------------------------

# Projects Overview

1. Movie Genre Classification
This project predicts the genre of a movie based on its plot description.
    
     - Converted text data into numerical form using TF-IDF
     - Trained a Logistic Regression model
     - Achieved accurate genre predictions based on input text

2. Spam SMS Detection
This project classifies SMS messages as Spam or Ham (legitimate).

    - Preprocessed text data
    - Used TF-IDF vectorization
    - Implemented Naive Bayes algorithm for classification
    - Tested model on custom messages

3. Handwritten Text Generation
This project generates text character by character using deep learning.

    - Implemented LSTM (Recurrent Neural Network)
    - Trained on sample text data
    - Generated new sequences based on learned patterns
